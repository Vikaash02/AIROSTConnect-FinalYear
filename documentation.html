class ProgramRecommender (programs=None)

ProgramRecommender is a class that provides a recommendation system for programs based on user-selected categories. It uses TF-IDF (Term Frequency-Inverse Document Frequency) and cosine similarity to recommend programs that match a user's preferences.

Initializes the ProgramRecommender class.

:param programs: A list of programs to initialize the recommender system. If no programs are provided, an empty list is used.

class ProgramRecommender:

    def __init__(self, programs=None):
        self.programs = programs or []
        self.vectorizer = TfidfVectorizer(stop_words='english')

    def add_program(self, program):
        self.programs.append(program)

    def generate_recommendations(self, user_categories, top_n=5):
    
        if not self.programs:
            print("No programs available.")
            return []

        descriptions = [
            f"{p.get('name', '')} {p.get('category', '')} {p.get('notes', '')}" 
            for p in self.programs
        ]
        
        tfidf_matrix = self.vectorizer.fit_transform(descriptions)

        category_matches = [
            p for p in self.programs 
            if p.get('category') in user_categories
        ]

        print(f"Matching programs for categories {user_categories}: {category_matches}")  # Debugging

        if not category_matches:
            print(f"No programs found for the selected categories: {user_categories}")
            return []

        category_descriptions = [
            f"{p.get('name', '')} {p.get('category', '')} {p.get('notes', '')}" 
            for p in category_matches
        ]

        category_tfidf = self.vectorizer.transform(category_descriptions)

        similarities = cosine_similarity(category_tfidf, tfidf_matrix)

        similar_program_indices = similarities.argmax(axis=1)

        recommended_programs = [self.programs[i] for i in similar_program_indices[:top_n]]
        
        return recommended_programs

def generate_recommendations(self, user_categories, top_n=5)

Generates program recommendations based on the user's selected categories using the TF-IDF vectorization and cosine similarity.

The algorithm performs the following steps: 1. Filters programs based on the user's preferred categories. 2. Converts both program descriptions and category-matched programs into TF-IDF matrices. 3. Calculates cosine similarities between category-matched programs and all available programs. 4. Returns the top N most similar programs based on cosine similarity.

:param user_categories: List of categories that the user has selected.
:param top_n: Number of recommendations to return (default is 5).
:return: A list of the top N recommended programs.
